== Description ==

woocommerce 플러그인의 한국 결제(paygate)를 추가합니다.

플러그인 사용을 위해서는 paygate측과 별도 계약이 성립되어야 하며, 관련 내용은 플러그인 설정 페이지에서 추가 되어야 적용됩니다.

API 인증값은 paygate측의 암호화 관련 부분을 사용하게 하며 보안을 위해 꼭 사용하실 것을 권장합니다.

API 인증값을 사용하지 않을 경우 최소한의 보안 처리만 가능하며 보안상 취약점이 생길 수 있습니다.
